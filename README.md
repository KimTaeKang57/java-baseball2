# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 기능 목록

### 경주 게임
시작한다
입력 된 회차가 끝나면 종료한다
### 자동차
0~9사이 무작위 수에서 4이상인 경우에 전진한다.
0~9사이 무작위 수에서 4미만인 경우에 정지한다.
입력한 숫자에 해당하는 자동차 이름이 부여된다.

## 사용자
숫자를 입력한다
이름을 입력한다
